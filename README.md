# barbara-saas-marketing

合同会社バーバラ企画 SaaSマーケティング戦略リポジトリ

## 概要

WebORCA API活用の患者離脱防止SaaS「バーバラSaaS」のマーケティング施策を管理するリポジトリです。

## ディレクトリ構成

| ディレクトリ | 内容 |
|------------|------|
| `CLAUDE.md` | エージェントチーム用プロンプト（プロダクトコンテキスト・チーム編成・タスク定義） |
| `articles/` | 完成記事のMarkdown |
| `calendar/` | 投稿カレンダー |
| `research/` | 競合分析・KW調査 |
| `assets/` | 画像指示書等 |

## 関連リポジトリ

- [barbara-saas](https://github.com/beru3/barbara-saas) — プロダクト開発（API・分析・レポート生成）
