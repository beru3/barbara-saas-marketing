## 各ページの設計

### 1. トップページ（index.html）

バーバラ企画の会社サイト。事業の1つとしてミルカルテを紹介。

| セクション | 内容 |
|-----------|------|
| **Header** | ロゴ（合同会社バーバラ企画）＋ナビ（事業紹介 / 代表紹介 / 会社概要） |
| **ヒーロー** | 会社のミッション・ビジョン。「医療現場のデータ活用を、もっと身近に」的なコピー |
| **事業紹介** | ミルカルテを紹介するカード。将来の事業追加に対応できる構成。→ /mirukarte へリンク |
| **代表プロフィール** | 古家聡大。複数クリニックのレセコン運用・データ管理の経験。顔写真枠 |
| **会社概要** | 社名・代表者名・所在地・連絡先 |
| **Footer** | コピーライト＋ナビリンク |

### 2. ミルカルテ紹介ページ（mirukarte/index.html）

ミルカルテのサービス訴求ページ。院長向け。

| セクション | 内容 |
|-----------|------|
| **Header** | トップと共通 |
| **ヒーロー** | 「来なくなった患者に、気づいていますか？」＋サブコピー＋CTA |
| **課題提起** | 院長が見落としている3つの数字（離脱人数・損失額・復帰可能性）。カード形式 |
| **サービス概要** | 発見→実行→測定の3ステップフロー図 |
| **競合比較** | 「分析で終わらない」比較表（CLINIC BOARD等 vs ミルカルテ） |
| **料金** | Tier 0〜4の概要。「まずは無料レポートから」の訴求 |
| **CTA** | 無料PDFレポートへ誘導（/lp/report へ） |
| **Footer** | トップと共通 |

### 3. PDFダウンロードLP（lp/report/index.html）

メールアドレスと引き換えにPDFレポートをダウンロードするLP。

| セクション | 内容 |
|-----------|------|
| **Header** | 簡略版（ロゴ＋トップへのリンクのみ） |
| **ヒーロー** | 「3ヶ月来ていない患者、何人いますか？」＋「無料レポートで離脱の実態を知る」 |
| **レポート内容** | PDFの中身を3〜4項目でプレビュー（離脱実態・損失計算・3つの習慣・改善事例） |
| **フォーム** | Brevoフォーム埋め込みプレースホルダ（`<div id="brevo-form-report">`） |
| **安心材料** | 「売り込みメールは送りません」「いつでも配信解除可能」 |
| **Footer** | 簡略版 |

### 4. モニター登録LP（lp/monitor/index.html）

無料モニターに申し込むLP。BOFU記事からの導線。

| セクション | 内容 |
|-----------|------|
| **Header** | 簡略版 |
| **ヒーロー** | 「先生のクリニックのデータで、無料レポートをお作りします」 |
| **受け取れるもの** | 離脱予備軍レポート・推定損失額・疾患別内訳・改善提案 |
| **お願いすること** | レセコン集計データ提供＋月1回15分フィードバック面談 |
| **流れ** | 申込→データ受領→レポート作成→フィードバックの4ステップ |
| **フォーム** | Brevoフォーム埋め込みプレースホルダ（`<div id="brevo-form-monitor">`） |
| **FAQ** | 費用は？→無料 / 期間は？→3ヶ月 / データは？→厳重管理 |
| **Footer** | 簡略版 |

## デザイン方針

| 項目 | 内容 |
|------|------|
| カラー | ネイビー(#1a2a4a) + 白(#ffffff) + アクセント青(#2563eb) + テキスト(#333) |
| フォント | system-ui（ゴシック系。Webフォント不使用で軽量化） |
| レスポンシブ | モバイルファースト（320px / 768px / 1024px / 1440px） |
| 画像 | 初期はCSSグラデーション＋Unicodeアイコンで代用。写真は後日追加 |
| JS | モバイルメニュートグルのみ（10行程度のインラインscript） |
