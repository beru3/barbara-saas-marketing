## p.2｜患者離脱の実態——「減っている実感がない」が最大の危険信号

「先生、この3ヶ月間、来院が途絶えている患者さんは何人いますか？」

私はこれまで複数のクリニックのレセコンデータに関わってきましたが、この質問に即答できた院長はほぼゼロです。

これは先生の怠慢ではありません。**そもそも、確認する仕組みがない**のです。

### なぜ気づけないのか——3つの構造的な理由

**理由1：新患が「穴」を埋めてしまう**

毎月の患者数が大きく減っていないように見えるのは、新患が来ているからです。しかしその裏で、かつて毎月通っていた慢性疾患の患者が1人、また1人と足が遠のいている。新患と離脱が相殺されると、表面上の数字は横ばいに見えます。

**理由2：レセコンは「来た人」しか見せてくれない**

レセコンの画面に出てくるのは、今日来院した患者、今月のレセプト対象者。「来なくなった人のリスト」は、意識的に抽出しない限り目に入りません。データはあるのに、見る仕組みがない。

**理由3：スタッフの感覚と実数のズレ**

受付スタッフに「最近来なくなった患者さん、いますか？」と聞くと、「田中さん、最近見かけませんね」くらいの答えが返ってきます。しかし実際にデータを引いてみると、感覚で把握している人数と実数には**3〜5倍の開き**があることが珍しくありません。

5人だと思っていたら20人。20人だと思っていたら60人。この差が、じわじわと経営を蝕んでいきます。

【デザイン指示】
- 右上にコールアウトボックス：「感覚で5人 → 実数で20〜30人。このズレが経営を蝕む」
- 3つの理由をアイコン付きで視覚的に分離

## p.3｜機会損失の計算——先生の院では年間いくらですか？

### モデルケース：内科クリニック（生活習慣病メイン）

| 項目 | 数値 |
|------|------|
| 全アクティブ患者数 | 600人 |
| うち慢性疾患（高血圧・糖尿病・脂質異常症） | 350人 |
| 3ヶ月以上来院なし（離脱予備軍） | **42人（12%）** |

> この「12%」という数字は特別に多いわけではありません。私がデータを見てきた複数のクリニックで、慢性疾患患者の10〜15%が「3ヶ月以上未来院」の状態にあります。

### 42人を放置するとどうなるか

1人あたりの月間平均診療単価を8,500円（850点）とすると：

```
42人 × 8,500円 ＝ 月 357,000円
357,000円 × 12ヶ月 ＝ 年間 約428万円
```

ただし、これは「42人全員が毎月来ていたら」の最大値です。現実的に計算してみます。

### 現実的な回収シミュレーション

| シナリオ | 復帰人数 | 月額回収 | 年間回収 | 施策コスト（目安） | 実質利益 |
|---------|---------|---------|---------|-----------------|---------|
| 控えめ（2割復帰） | 8人 | 6.8万円 | **82万円** | 約12万円 | **約70万円** |
| 標準（3割復帰） | 13人 | 11万円 | **133万円** | 約18万円 | **約115万円** |
| 積極的（5割復帰） | 21人 | 18万円 | **214万円** | 約30万円 | **約184万円** |

> **施策コストの内訳（目安）：** はがきDM（1通80〜100円×対象人数×年数回）＋LINE公式アカウント運用（月額0〜5,000円）。新患獲得のWeb広告（1患者あたり5,000〜15,000円）と比較すると、1/10以下のコストです。

### 新患獲得 vs 離脱患者の復帰——コスト比較

| | 新患獲得 | 離脱患者の復帰 |
|---|---|---|
| 1人あたりの獲得コスト | 5,000〜15,000円 | 80〜500円（はがき1通） |
| 信頼関係 | ゼロから構築 | すでに構築済み |
| 初回の心理的ハードル | 高い | 低い（一度来たことがある） |
| 平均継続率 | 不安定 | 既存患者のほうが高い傾向 |

**新患を増やすことが悪いわけではありません。しかし、「一度来てくれた患者を大切にする」ほうが、圧倒的にコスト効率が高い。** 多くの院長がここを見落としています。

## p.4｜離脱を防いでいるクリニックの3つの習慣

「うちもやらないと」と思った先生へ。特別なことは必要ありません。離脱率の低いクリニックに共通するのは、次の3つの習慣だけです。

---

### 習慣1：月に1回、「来ていない人」を数える

飲食チェーンの店長は、毎日の客数・客単価・リピート率を見ています。美容院のオーナーは、3ヶ月来ていない顧客にDMを打つのが当たり前です。フィットネスジムは、休眠会員の掘り起こしに専任スタッフを置いています。

ところがクリニックでは、「3ヶ月来ていない患者が何人か」を月次で確認している院長はほとんどいません。

やることはシンプルです。

> **月初に、「最終来院日が3ヶ月以上前の慢性疾患患者」の人数を確認する。**

これだけで、離脱の「見える化」が始まります。先月より増えたか減ったか。この推移を追うだけで、経営の手触りが変わります。

---

### 習慣2：離脱予備軍に「声をかける」仕組みを持つ

数字を把握しても、見ているだけでは患者は戻りません。大事なのは**「声をかける」アクション**です。

ただし、これを院長やスタッフの善意と気合いに頼ると続きません。「先月の未来院リストから、はがきを送る」——この動作が**月次の業務フローに組み込まれているか**が分かれ目です。

| 方法 | 向いている患者層 | コスト/通 | 開封・閲読率 | 運用負荷 |
|------|---------------|----------|------------|---------|
| はがき・手紙 | 60代以上、高齢者 | 80〜120円 | 高い（80%以上） | 中（印刷・発送） |
| LINE | 40〜50代、スマホ利用者 | ほぼ0円 | 中（60%前後） | 低（テンプレ配信） |
| 電話 | 重要な患者（高単価・長期） | 0円 | 最も高い | 高（1件3〜5分） |
| SMS | 幅広い年齢層 | 8〜15円 | 高い（90%以上） | 低 |

> **ポイント：** 全員に電話する必要はありません。まずは「はがき1枚」から始めるクリニックが多いです。「先生、最近お変わりないですか。お薬の件もありますので、お時間のある時にお越しください」——この一言で戻ってくる患者は想像以上にいます。

---

### 習慣3：やった結果を「数字」で振り返る

はがきを30通送った。そのうち何人が再来院したか。再来院した患者の診療単価はいくらだったか。はがき代と診療報酬の差額はいくらか。

この振り返りをしているクリニックは、**翌月の打ち手の精度がどんどん上がっていきます。**

- 「高血圧の患者はDMの反応がいいが、脂質異常症は反応が薄い」
- 「3ヶ月離脱より、6ヶ月離脱のほうが復帰率が低い。早めに動くべき」
- 「はがきよりLINEのほうが40代には効く」

こうした知見が自院のデータから積み上がっていく。これは**他院には絶対に真似できない、自院だけの経営資産**です。

【デザイン指示】
- 3つの習慣をステップ図（①②③）で表現
- 各習慣にアイコンを配置（①虫眼鏡 ②手紙 ③グラフ）
- はがき文面の例をコールアウトボックスで囲む
- ページ全体を通して「難しくない」「すぐできる」の印象を与える

## p.5｜「分かっている、でもできない」——その壁を越えるために

ここまで読んで、「そんなことは分かっている」と思った先生もいるかもしれません。

問題は「知っているかどうか」ではなく、**「毎月、実際に回せるかどうか」**です。

### 現実の壁：院長が1人でやると何が起きるか

仮に離脱予備軍への対応を院長が自分でやるとします。

| 作業 | 所要時間（月あたり） |
|------|-------------------|
| レセコンからデータ抽出・リスト作成 | 2〜3時間 |
| 対象患者の選定・文面作成 | 1〜2時間 |
| はがき印刷・宛名書き・投函 | 2〜4時間（30通の場合） |
| LINE配信の設定・文面調整 | 1〜2時間 |
| 再来院の集計・効果測定 | 1〜2時間 |
| **合計** | **月7〜13時間** |

月7〜13時間。診療の合間にこれを毎月続けるのは、現実的ではありません。

### スタッフに任せる？——属人化の罠

「じゃあスタッフに任せよう」と考えるのは自然です。しかし——

- **できるスタッフが辞めたら止まる**。引き継ぎ資料は大抵作られない
- **人によってやり方がバラバラ**。今月は30通、先月は10通、その前は0通
- **効果測定まで手が回らない**。「送りました」で終わり、結果を追わない

属人的な運用は、最初の3ヶ月は動きます。しかし半年、1年と続く仕組みにはなりません。

### 必要なのは「仕組み」

離脱対策が続いているクリニックに共通するのは、**特定の誰かが頑張っている**のではなく、**仕組みとして回っている**ということです。

```
毎月自動でデータを抽出
　　↓
対象リストが出てくる
　　↓
テンプレートに沿って声かけ（はがき or LINE）
　　↓
結果が自動で集計される
　　↓
翌月に改善が反映される
```

この流れが「勝手に回る」状態を作れるかどうか。それが、離脱対策が継続するかどうかの分かれ目です。

【デザイン指示】
- 月7〜13時間を時計のビジュアルで表現
- 「仕組み化」のフロー図は矢印付きで視覚的に
- ページの雰囲気は「共感」→「解決への橋渡し」。責めない、煽らない

## p.6｜ミルカルテ——「発見 → 実行 → 測定」を先生の代わりに回します

**ミルカルテ**は、外来クリニック向けの患者離脱防止サービスです。

p.4でお伝えした3つの習慣——「数える」「声をかける」「振り返る」。この一連の流れを、先生やスタッフの手を煩わせずに回す仕組みを提供します。

### ミルカルテが担う3ステップ

| ステップ | ミルカルテがやること | 先生がやること |
|---------|-------------------|-------------|
| **① 発見** | レセコンデータから離脱予備軍を自動検出。人数・推定損失額・疾患別の内訳をレポートで報告 | レポートを月1回確認する（5分） |
| **② 実行** | 対象患者へのはがきDM作成・印刷・発送を代行。LINE公式アカウントの配信設定・運用も対応 | 文面を確認・承認する（10分） |
| **③ 測定** | 施策後の再来院数・回収額・施策コストの損益を月次レポートで報告。翌月の改善提案も添付 | レポートを見て判断する（5分） |

**先生の月あたりの作業時間：約20分。**

### 「分析して終わり」のサービスとは違います

| | 一般的な分析サービス | ミルカルテ |
|---|---|---|
| 離脱患者の可視化 | あり | あり |
| 疾患別・期間別の詳細分析 | 一部あり | あり |
| 具体的な打ち手の提案 | なし or 一般論 | **先生の院のデータに基づく具体策** |
| はがきDMの代行 | **なし** | **あり（印刷・発送まで）** |
| LINE配信の運用代行 | **なし** | **あり（設定・配信まで）** |
| 施策の効果測定レポート | なし | **毎月提供** |
| 翌月の改善提案 | なし | **毎月提供** |

> 分析ダッシュボードを渡されても、見る時間がない。改善策を提案されても、実行する人がいない。——この「分かったけど動けない」問題を、ミルカルテは**実行の代行**で解決します。
